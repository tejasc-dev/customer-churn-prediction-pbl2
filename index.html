Customer Churn Prediction using Machine Learning

Predicting telecom customer attrition using supervised machine learning models to enable proactive retention strategies.

Problem Statement

Telecom companies face significant revenue loss due to customer churn. The objective of this project is to develop and compare multiple machine learning models to accurately predict customers likely to discontinue services.

Literature Review

Prior research shows ensemble methods such as Random Forest and Gradient Boosting outperform linear models in churn prediction tasks.

Research Gap

Limited comparative evaluation across multiple classifiers on the IBM Telco dataset with focus on churn recall.

System Methodology

Dataset

IBM Telco Customer Churn dataset (~7043 records). Data preprocessing and encoding applied.

Models Used

10 classifiers including Logistic Regression, Random Forest, Gradient Boosting and XGBoost.
