Introduction

Customer churn refers to customers discontinuing a service. In telecom industries, churn directly impacts revenue and growth.

Problem Statement

Traditional models fail to capture complex customer behavior. Class imbalance leads to biased predictions.

Objectives

  • Build churn prediction model
  • Compare ML algorithms
  • Use ensemble learning
  • Improve prediction accuracy

Methodology

  • Data preprocessing
  • Feature engineering
  • Model training
  • Ensemble learning
  • Evaluation

Models Used

  • Logistic Regression
  • Decision Tree
  • Random Forest
  • Gradient Boosting
  • Ensemble Model

Conclusion

Ensemble learning improves churn prediction by capturing complex patterns.

Future work includes deployment and deep learning models.
