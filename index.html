Customer Churn Prediction

Using Ensemble Learning Techniques


Tejas Chaudhary

2427030729

Manipal University Jaipur

Introduction

Customer churn refers to customers discontinuing a service. In telecom industries, churn directly impacts revenue and growth.

Problem Statement

Traditional machine learning models fail to capture complex customer behavior. Class imbalance leads to biased predictions.

Objectives

  • Develop churn prediction model
  • Compare machine learning algorithms
  • Implement ensemble techniques
  • Improve prediction performance

Methodology

  • Data preprocessing
  • Feature engineering
  • Model training
  • Stacking ensemble approach
  • Evaluation using metrics

Dataset

Telecom customer dataset including tenure, services, billing, and demographics.

Target Variable: Churn (Yes/No)

Models Used

  • Logistic Regression
  • Random Forest
  • XGBoost
  • LightGBM
  • Stacking Ensemble

Results

Stacking ensemble improved recall and overall performance compared to individual models.

Conclusion

Ensemble learning enhances churn prediction accuracy and robustness. Future work includes deployment and further optimization.
