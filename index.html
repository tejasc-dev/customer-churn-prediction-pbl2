Customer Churn Prediction

Using Ensemble Learning Techniques


Tejas Chaudhary

2427030729

Manipal University Jaipur

Introduction

Customer churn refers to customers discontinuing a service. It directly impacts revenue in telecom industries.

Problem Statement

Traditional machine learning models fail to capture complex customer behavior. Class imbalance leads to biased predictions.

Objectives

  • Develop churn prediction model
  • Compare ML algorithms
  • Use ensemble learning
  • Improve prediction performance

Methodology

  • Data preprocessing
  • Feature engineering
  • Model training
  • Stacking ensemble
  • Evaluation

Dataset

Telecom customer dataset with features like tenure, services, billing, and demographics.

Target: Churn (Yes/No)

Models Used

  • Logistic Regression
  • Random Forest
  • XGBoost
  • LightGBM
  • Stacking Ensemble

Results

Ensemble models outperformed individual models by capturing complex patterns.

  • Improved Accuracy
  • Better Recall
  • Stable Predictions

Conclusion

Ensemble learning significantly improves churn prediction performance.

Future work includes deployment and optimization.
