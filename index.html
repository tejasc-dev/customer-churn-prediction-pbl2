Introduction

Customer churn refers to customers discontinuing a service. In telecom industries, churn directly impacts revenue and growth.

Problem Statement

Traditional machine learning models fail to capture complex patterns in customer data. Class imbalance leads to biased predictions.

Objectives

  • Predict customer churn
  • Compare machine learning models
  • Implement ensemble techniques
  • Improve prediction performance

Methodology

  • Data preprocessing
  • Feature engineering
  • Model training
  • Stacking ensemble
  • Evaluation

Models Used

  • Logistic Regression
  • Random Forest
  • XGBoost
  • LightGBM
  • Stacking Ensemble

Results

The stacking model improved recall and overall performance compared to individual models.

Conclusion

Ensemble learning improves churn prediction accuracy and provides better generalization.
