Introduction

Customer churn refers to customers leaving a service. It directly impacts revenue.

Problem Statement

Traditional models fail to capture complex patterns and class imbalance issues.

Objectives

  • Predict churn
  • Compare models
  • Use ensemble learning
  • Improve accuracy

Methodology

  • Preprocessing
  • Feature engineering
  • Model training
  • Stacking ensemble
  • Evaluation

Models Used

  • Logistic Regression
  • Random Forest
  • XGBoost
  • LightGBM
  • Stacking

Results

Stacking model improved performance and recall significantly.

Conclusion

Ensemble learning improves churn prediction accuracy and reliability.
