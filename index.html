Customer Churn Prediction

Using Ensemble Learning Techniques

Tejas Chaudhary | 2427030729

Manipal University Jaipur

Introduction

Customer churn refers to customers discontinuing a service. In industries like telecom, churn directly impacts revenue and growth.

Predicting churn is a complex classification problem involving customer behavior, usage patterns, and service interactions.

Problem Statement

Traditional machine learning models struggle to capture complex patterns in customer data. Additionally, class imbalance leads to biased predictions.

The goal is to build a robust and accurate churn prediction system using ensemble learning.

Objectives

  • Develop a churn prediction model
  • Compare multiple machine learning algorithms
  • Improve performance using ensemble techniques
  • Analyze model performance using evaluation metrics

Methodology

The project follows a structured pipeline:

  • Data preprocessing and cleaning
  • Feature engineering
  • Model training
  • Ensemble learning
  • Evaluation and comparison

Dataset

The dataset contains customer information such as tenure, services used, billing details, and demographics.

Target Variable: Churn (Yes/No)

Models Used

  • Logistic Regression
  • Decision Tree
  • Random Forest
  • Gradient Boosting
  • Ensemble Model

Results and Discussion

Ensemble models outperformed individual models by capturing complex patterns in customer behavior.

  • Improved Accuracy
  • Better Recall for churn class
  • More stable predictions

Conclusion

Ensemble learning significantly improves churn prediction performance. The model can be used by businesses to proactively retain customers.

Future work includes deploying the model as a web application and improving performance with deep learning.
