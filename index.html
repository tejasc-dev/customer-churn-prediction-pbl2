Customer Churn Prediction

Using Ensemble Learning


Tejas Chaudhary

2427030729 | MUJ

Introduction

Customer churn refers to customers leaving a service. In telecom industries, churn directly impacts revenue.

Problem Statement

Traditional models fail to capture complex patterns. Class imbalance leads to poor churn detection.

Objectives

  • Predict customer churn
  • Compare ML models
  • Use ensemble learning
  • Improve accuracy

Methodology

  • Data preprocessing
  • Feature engineering
  • Model training
  • Stacking ensemble
  • Evaluation

Models Used

  • Logistic Regression
  • Random Forest
  • XGBoost
  • LightGBM
  • Stacking Ensemble

Results

Stacking model improved performance significantly. Better recall for churn class achieved.

Conclusion

Ensemble learning improves prediction accuracy and reliability. Future work includes deployment and optimization.
