Introduction

Customer churn refers to customers leaving a service. It directly impacts revenue in telecom industries.

Problem Statement

Traditional models fail to capture complex patterns. Class imbalance leads to inaccurate predictions.

Objectives

  • Predict customer churn
  • Compare ML models
  • Use ensemble learning
  • Improve performance

Methodology

  • Data preprocessing
  • Feature engineering
  • Model training
  • Stacking ensemble
  • Evaluation

Models Used

  • Logistic Regression
  • Random Forest
  • XGBoost
  • LightGBM
  • Stacking Ensemble

Results

Stacking model improved performance significantly. Better recall for churn achieved.

Conclusion

Ensemble learning improves prediction accuracy. Future work includes deployment.
